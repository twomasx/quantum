About the Project

Q.U.A.N.T.U.M. is a consciousness research platform designed to explore the potential link between human intention, quantum randomness, and the measurable effects of consciousness on physical systems.

All experiments run locally in your browser using cryptographically secure random number generation, ensuring no external manipulation of results.

How to Use

Each experiment includes both batch and single modes:

  • Batch Mode: Run a large number of trials automatically to gather statistical data.
  • Single Mode: Perform one trial at a time with focused attention for stronger intentional effects.

All experiments include a control group simulation to help differentiate true effects from statistical coincidence.
